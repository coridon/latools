.. _install:

************
Installation
************

.. only:: html

    :Release: |version|
    :Date: |today|

Download `latools-X-X-X.tar.gz <https://drive.google.com/drive/folders/0B0R-S9phfYhmRm1VN3B0c2dsUjg>`_, and save it to :code:`convenient_folder`.

Open a terminal window, :code:`cd` into :code:`convenient_folder`, and run:

.. code-block:: bash

	pip install latools-X.X.X.tar.gz

If this is your first time, read through the :ref:`getting_started` guide. Otherwise, you're good to go!
